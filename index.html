What is this

Dotfiles are configuration files that control your shell, git, editor, and dozens of other tools.

This project is a tested, optimized collection — not random configs from the internet, but a coherent setup where everything works together.

Performance

5x faster shell startup

Most zsh setups load everything at startup. We use lazy loading — tools initialize only when you use them.

What's included

Everything you need

Sync across machines

Chezmoi keeps your dotfiles in sync. Change something on your laptop — it's on your server in seconds. Templates handle machine-specific differences.

Fast plugin loading

Sheldon manages zsh plugins without bloat. Written in Rust, it loads plugins in parallel and defers non-essential ones until after your prompt appears.

Smart prompt

Starship shows git status, directory, language versions — but only when relevant. No clutter, no lag, works everywhere.

Version management

One tool for all runtimes. ASDF handles Node, Python, Ruby, Go, Rust — per-project versions via .tool-versions.

Modern CLI tools

eza replaces ls, bat replaces cat, ripgrep replaces grep, fd replaces find. Faster, prettier, actually usable.

Smart navigation

Zoxide learns your habits — type z proj to jump to ~/work/projects. fzf adds fuzzy search to files, history, branches.
