What is this?

Dotfiles are configuration files that live in your home directory. They control how your shell looks, what aliases you have, how git behaves, and hundreds of other settings.

This project is a curated, tested, and optimized collection of dotfiles. Instead of copying random configs from the internet, you get a coherent setup where everything works together.

What's Included

Configuration Management

Your dotfiles sync across machines with Chezmoi. Change something on your laptop ‚Äî it's on your server in seconds. Templates handle machine-specific differences automatically.

Fast Plugin System

Sheldon manages zsh plugins without the bloat. Written in Rust, it loads plugins in parallel and defers non-essential ones until after your prompt appears.

Smart Prompt

Starship shows git status, current directory, language versions ‚Äî but only when relevant. No clutter, no lag, works in any terminal.

Version Management

One tool for all runtimes. ASDF handles Node, Python, Ruby, Go, Rust ‚Äî whatever you need. Per-project versions via .tool-versions files.

Better Defaults

Modern replacements for ancient Unix tools: eza instead of ls (with colors and git status), bat instead of cat (with syntax highlighting), ripgrep instead of grep (10x faster), fd instead of find (actually usable).

Smart Navigation

Zoxide learns your habits. Type z proj and jump to ~/work/projects because that's where you always go. fzf adds fuzzy search to everything ‚Äî files, history, git branches.
